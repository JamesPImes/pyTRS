
``PLSSDesc``
============

Use this class to break full PLSS land descriptions into tracts, and
to extract or export the resulting data in bulk.

(Implemented at ``pytrs.parser.plssdesc.PLSSDesc`` but automatically
imported as a top-level class, ``pytrs.PLSSDesc``.)
